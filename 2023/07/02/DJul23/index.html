七月随笔

发表于 2023-07-02 更新于 2023-07-21 分类于 水木录
“七月的七海千秋”

07/01

七月伊始，今日情绪稳定，大概。但是缺觉。写这个的时候拉练的学生应该刚出发，希望早上回来都能睡个好觉。

今天好像没什么写的，随便记点流水账吧。

早上开党团生活会，其实就是个批评会，进行批评与自我批评。平时随笔里自我批评也挺多了，会上也差不多就是这个样子。但是，但是！最尴尬的是我们班主任是负责党支部的，然后我讲了许多我自己的和我们班的问题，最后发现他在线上全程都在听，寄。

下午忙前后端去了。因为不敢乱动代码，写异步写了一下午，看 React 调 CSS 又花了一晚上。最后弄得还可以，接下来想学习一下怎么弄整个的布局。作为一个对 UI 还有那么点要求的人来说弄前端可能是一种折磨的享受，但是感觉比调后端的过程更舒服一点（虽然个人感觉后端还好写一些）。总之被委派做美工负责人了（x

07/02

从明天开始大概就不会在现在这个点（指11-2点）看到我了，因为我终于要调作息了！作为前端人被安排在深夜工作，所以要早睡早起了，听说是天安门国旗仪仗队的作息，太好了，深夜我与祖国同在！

去报了暑校的辅导员。前段时间去招生的时候第一次知道原来还有暑校这种活动，估计是搞竞赛停课的锅，我一直只知道有个寒校（挑战赛）。 还挺好奇暑校是什么样的一种活动。老实说我也不知道为什么我会去报名，可能是因为舍友也去报了，顺便想给自己寡淡的简历增添一点东西，估计到时候被刷下来的就是我吧！（自我陈述里写自己乐观开朗等等，看过前面随笔的都知道不完全真实，自己想成为的样子都写在陈述里，至于究竟是不是，如是。但是均绩肯定是是真实的。）

写前端有一段时间了，感觉任务量比后端艰巨很多。一想到我们之后要实现的各种功能，还有可能要慢慢调的布局之类的 UI 设计，只剩下不到四天（没扣掉上课）去实现，有点麻。

军训相关：去年因为拉练买了一床被子，一直放在宿舍，前段时间拿给甜柠檬用。因为当时两横三竖没拆掉，据甜柠檬说：

您的被子真好用，宿舍里三个人都拿来打的背包（

然后拉练休息的时候我还没睡，稍微聊了一会。意识到一件生草的事情：可以一直把被子传给后辈，并表示：这是我们学校祖传的被子！ 但是给后来人省掉很多钱，何乐而不为，还可以学会两横三竖，挺好的。

07/03

既然这段时间我都是下午六七点开始写今日随笔，那是不是可以取个名字，比如叫《华清晚报》之类的

学弟这两天来华子参加羟基面试，存了一箱芒果给我，出校去拿了。每次踩车后面放着大箱子都会蹑手蹑脚，好害怕箱子突然侧滑翻出去，这次踩了好几公里，比去时的速度慢了不少。经过六教门口有个羟基的标语，其实还挺好奇羟基面试会是什么样的。

朋友圈今天被“网络暴力下的真实清华”填满了。感觉有很多可以展开说的东西，比如理性爱国，比如控评，但是最近忙，先不写了，也许以后有机会再写。还想起家乡今年高考结束开始填志愿时流言漫天，因为各种谣言朋友圈和空间里到处都弥漫着非常激动的情绪。前端时间还看到有篇文章说“互联网已经被AI污染了”，未来谣言越来越多，网络环境实际上也处在过激的边缘，有办法去遏制这样的景况么？

07/04

今天被 css/eslint 和不知道谁加的最大警告限制整破防了，所以今天不写了。（最好笑的是之前样例代码还有个“你可以加入css文件”之类的注释，呵，全是骗人的）

此时凌晨3点，按之前的+12时区我通宵了（

07/05

九点半起，昏昏欲睡。上前后端。问如何解决警告，老师表示这种导入方式不好，应该写 Typescript 里面，研究的动画无了，大悲。课终于讲到前端，作为只会打 .css 的前端工程师被称“已经过时了”“不如函数式编程”云云，惨，我们前端工程师怎么你了 TAT 没有什么意思，主要是忙了那么久听到这句话当时挺破防的 后面又花了不少时间把 .css 搬运到 .tsx 里面打包。下午真正开始系统地学 React，发觉 React 函数式编程确实挺强大的。不过像 .css 这样的 dirty work 现在还是得要自己精细实现才行啊。虽然实现得也不好就是了。

荷园纠结钱包，桃李纠结食物。荷园红烧肉真做得挺好，感觉可能比熙春园还好吃。

07/06

（07/09：中间这三天太伤身了，所以现在才来补。没有任何针对或者指向。）

晚上很困，本来打算休息一会慢慢肝美工。然后，然后，组里面发现了不少的 Bug ……干了那么久的前端美工，有点体会到 New Game 里面程序组的那种被改需求的绝望了。而且，不要以为样式设计随便写几行就结束了啊啊啊啊啊啊啊啊 TAT

总之一个人从晚九弄到了早六，算是连续工作了 25h，破纪录了。后面感觉人虚弱了许多，总是想睡觉。（所以能不熬夜就不要熬夜。）好消息是弄了许多比较好用的函数式的组件，而且非常简洁好看，之后重新改版面的时候还可以继续用。

07/07

睡了一早上。然后下午忙着支队报销的事情。

今年出去社会实践，组长正好是我们团支书。报名的时候跟他聊，他说还没有人管财务。我说，啊这样啊，早知道我填财务了。真傻，竟然忘了会计审计的工作会累死累活。 然后就开始按策划那边的要求订酒店。虽然收了队费看着卡上的钱，什么大富翁，但是不太希望轻易动这笔帐，于是跟酒店协商好了院里填报销单子。这单子看着就烦……填得慢，而且审批还慢，一套流程下来代计课也没听成，已经落下了两节半的课，完蛋了。 晚上想了一晚上的 DP 没想出来，随便交了个东西上去应付代计，摆了，但是难受。

07/08

下午去参加暑校辅导员的面试（前情请见07/02）。早上摸鱼，啥也没准备，光看清华历史了。午饭意识到不好了，快速赶完一篇自我介绍，草草背了一下，最后果然也没完全背下来。

坐一排的有研一，有直博，有经管/车辆大三，然后一听自我介绍，都不是一个画风，看不出来像是准备的，像是真情流露，社工经验非常丰富，基本都当过辅导员。然后我颤颤巍巍地坐在正中央，自我介绍的时候声音在发抖哈哈哈哈哈哈哈哈，还是太稚嫩了。好处是因为正中间，所以顺序回答问题怎么着也能有个大概的参考。总而言之表现差强人意，就当作是来学习经验了；算是学到了很多面试，还有做辅导工作的经验。

如果有人荷塘落水了，你怎么做？

只要会游泳（还有人有深水证的，不过深水证应该在座的考就有）

几个人想了很多做法，比如旁边医院啊，做预案啊，大声呼救看看有没有专业人员/四处寻找可用的救助工具，实在不行再下去救，人员安置到比较安全的地方等等。最后一位回答的比较好，一是会游泳不等于会救，二是一个人做不完上述这么多工作，要能分工。

你认为做辅导员最重要的是哪三个方面？

这个看个人选择吧。我印象里回答的是“组织策划”，“危机应对”和“立场/底线意识”。（其实这个回答太 leader 了，没有人情味。）其他的一些回答包括热情（很正确，不过和答者说的一样，是辅导员最基本要有的）、时间管理、安全意识（很重要，当时没考虑到），还有清华人文风物的知识储备等等。我感觉热情不如换成“主动”更好一些，主动式的热情比起被动式的更有效果一些；当然也要照顾学生的感受，要考虑这样的行为别人会怎么想。

如果有学生闹着要回家，你怎么解决？

大家回答基本是一样的，先了解学生这么做的原因，然后有一些缓冲举动，最后根据实际情况再做进一步的措施。缓冲举动这个每个人有不同的回答吧，有的会妥协/让步，有的会安抚情绪之后还是要求跟随大部队行动（主流）。总之见仁见智（

晚上回来又写代计到很晚，麻。

07/09

今天没啥事，除了很疲倦之外好像没啥别的了。还是得早点睡觉。

07/10

暑校辅导员的面试果然没过。那就随他去吧，可以早点回家也挺好的。晚上等消息的时候终于回味过来我的社恐属性，反而开始不想通过了。

昨晚虽然说早点睡但是熬了个通宵，又是赶美工。早上 code review 似乎反馈还行，老师提的改进意见好像基本没落在美工上（至少没在我负责的那几个部分）。

下午舍友去降旗，在大礼堂周边待了半小时。去工字厅转了一圈，荷花还没开盛。又绕到没走过的科学馆背后，发现有几张白色的高脚椅，远远看去以为是隐秘的露天咖啡馆。漫无目的地走啊走，感觉……生活好像本来应该是这样子的。我自己对这个园子了解多少呢？说不定还不如游客了解得多。平时从来没有为了转悠而转悠，也很少在衣食上要求什么，生活未免太浅薄了一些。

本来还想说点什么的，想想还是妥协吧。

07/11

为啥一直没写随笔？因为调时区了 意识到自从开始弄前后端以来就一直阴间作息，睡觉从来没睡够过。到后面不到天亮我甚至睡不下去，完了，这下真的完全夜猫子了。生活就变成了早上睡到中午，中午起来上课/调前后端，下午摸鱼，晚上通宵前后端。讽刺的是因为我负责 UI，保存下来的代码量不是很多（见后面），跟组长谈到后面 report 的时候被要求再多干一点活了，要不然 CoL 可能不够。 我对自己代码质量还比较满意，组里面前端不少重大 Bug 都是我在修，可是为什么最后不剩几行代码呢 TAT

07/12

前后端一晚上。听组长说我们现在这个 Layout 要拿不到4.0了，啊？（我觉得这个很精髓，没有这个要字我估计摆了。）然后看了外部库（之前是自己手写的 css），通宵了一晚上换了一整个 UI。

07/13

通宵：等上床已经早九了，根本没睡够，埋下了下午的伏笔。

下午继续修前后端，五点终于是修好了我们知道的所有的 Bug。可惜 Demo 已经录了，没能用上新 UI，但是我觉得比较离谱的是早上拿来做 Demo 的那一版竟然是之前带了各种 Bug 的版本，没用昨晚修好的。麻。 这时候我还没看过 Demo，想着晚上如果复习完代计有机会再重新弄吧，遂上床睡觉。因为这两天下雨不好穿着衣服睡觉，太困了也没换睡衣，直接脱掉上衣躺了。

六点起来，躺着看手机，发现初中班主任和某老同学打了几个语音通话，惊恐，赶紧先给同学打电话回去。刚开始似乎网络问题没有接通。然后又打了视频电话过来；我因为还没睡醒，朦朦胧胧地点了接听，然后初中老师出现在了画面上……等我接完电话过了几分钟,啊啊啊啊啊啊啊啊啊啊啊啊啊啊啊啊啊啊啊啊啊啊啊啊啊啊啊啊啊啊！宿舍里充满了哀嚎，后悔已经来不及了。被叫去科技园吃饭训话了，悲。在餐桌上不知道聊点什么，被吐槽没有生活情趣了。唉，确实是这样的。被嘱咐了大学生活一要谈一场恋爱坏了，怎么又是这个话题，二要检验自己选择的方向，三要有一次真正的实践（好像解释过是要接一份项目/出去实习）。

回来太晚，Cheat sheet 也没做，好在舍友给了一份，感激。

07/14

终于考完试，基本算是放假了。快乐北冰洋。

07/15

打了半天粥粥，还有半天在睡觉。肉鸽难玩，放了各种恶心的东西，但似乎因为 BGM 挺入耳，玩两天暂时还没腻。

中午外卖订了螺蛳粉，因为前段时间舍友对柳州粉感兴趣。翻了半天没找到螺蛳，螺蛳粉最重要的是螺蛳，用来提鲜味的；但好像没人在乎螺蛳。而且不管是量还是味道都感觉不够，因为原版螺蛳粉的辣度给我留下过心理创伤特意点了微辣，没想到还真的是微辣……这时候又开始想回去嗦粉（吃粉）。

虽然不能轻易冠之“粉之都”的名号（一家卖桂林米粉的连锁店，现在感觉没以前好吃了），广西与粉确实有着很深的渊源。不同地方有不同的粉。南宁因为文化底蕴不够深厚，历来被别几个著名旅游城市所轻薄，但也有独特的老友粉；邕宁区、蒲庙镇有生榨米粉。桂林是著名的桂林米粉，柳州是著名的螺蛳粉。其他的市、县乃至乡、屯还有不同的地方特色粉（比如全州的红油米粉、马山的牛肉粉等等。）猪杂、牛杂、猪肉粉，切粉、圆粉、粉利，干捞、水煮、烫粉，米粉、红薯粉、藕粉等等等等，每个地方都有不同的料、不同的做法、不同的粉，实在很难一一说清。

这些粉相较于其他地儿总有一种独特的酸味，有的浓烈（比如螺蛳粉，不过那个味道主要是酸笋的酸臭），有的淡到入口才能品尝出来。这种酸味，来自于米粉的发酵。假若到蒲庙去点一份米粉，你看着四五十岁的女店主操作着钢制的机器压着面团，面顺着底下的小孔压成圆粉状进到开水里慢慢地烫开，水上泛起一层淡白色的泡沫。不同地方的粉尝起来也不一样。有的地方吃起来很有弹性，靠手上两双木筷也很难夹断，吃起来只感觉一节一节的米粉在喉咙里蹦跶（一般是圆粉）。有的地方则做得软一些，吃起来有一种糯玉米的糯感（一般是切粉/粉利，但粉利做得很厚，吃起来不全是这种感觉，有点像是不粘牙的薄年糕）。至于粉里加的花生、葱花、紫苏、香菜、辣椒碎/泡椒、酸豆角等等，都是在境内不管到哪里去总能找见的配料；他们看着只是添头，但却与粉、肉（和汤）一起构成了一个广西人餐桌上的一碗粉。

哇怎么写了这么多，还是收手吧（

晚上前后端小组另一位来宿舍调代码，然后两个人都在拿平板打 Phigros，好活，音游浓度又变高了。不对啊，不是100%到100%，没变吗。本来大家还在为项目分发愁，没想到老师找上门来表示院里看着感觉不错啊？看着我手写的那个旧 UI 还觉得不错？，你们愿意的话就把系统部署一下吧。总之我们好像要继续负责这个项目一直到九月份，完了，这下又少了摸鱼时间，惨。 但是实话实说真的很有成就感，有一种看着孩子长大了考上北大的感觉（笑）。可惜我没写多少代码，希望老师能信守诺言给个 A，别扣我代码分了。中午看到有个 UI 设计和前端工程师的对比视频蚌埠住了，原来我一直是兼任两职，这两个竟然还能分开…… 院里有人说项目写完后扔掉“有一种神秘的失落感”，大家都投入了许多时间、精力与心血，无论是被甲方选中后的激动还是没被选中/选择院里没有需求的项目后准备丢弃的感伤，都证明了这一点。

所以我还是觉得明年还是不要让 GPT 来全生成代码比较好，至少先让学生自己学明白语言。GPT 生成的还是烂了点，而且 UI 设计这块一点也不人性。

07/16

准备放假了我的作息还没完全调过来，还是很晚才睡很晚才起，吃饭两餐有时候甚至缩减为一餐。留着肚子等着去实践的时候好好享受。 下午忙着导项目到清华 gitlab 上，还不知道怎么处理 CI/CD 的 runner。依着留言去弄了 Jetbrain 的 Student Pack，这下组里不用担心后面运维问题了。

晚上舍友从暑校回来了，还有两个来宿舍里做客（打 Phigros，三个人三个平板，这不是音游杯吗？），一时间宿舍里非常热闹。只是好像跟我预想的假期氛围格格不入（

07/17

今天没做什么，休息了一天。仔细算下来玩了1/4天音游1/4天肉鸽。还有一点时间在摆。

院里财务处理好慢，老师说是因为放暑假了，什么，财务放暑假吗？ 可能说的是值班的人比平时少吧（

07/18

（07/19：确实是同一个人，惋惜。我不想在这里再多写更多东西，让未来的我回忆时还要去翻掘；也不计划为此做出更进一步的举动，亡者值得安宁的归所。我唯一在思考的是，假若她不同是茶园人，不同是闭社社友，面对一个我不认识的人，是否还能有如此深的感慨。）

一些也许不太成熟的想法：每次听闻这样的事情总是不自觉的一番慨叹与惋惜，但同时经历过各种各样的事情后，我尊重着 TA 们的决定，因为大多数时候，他们也清楚这样行动的代价与需要付出的勇气；同时也接受自己的弱小，明白自己无力去拯救每一个身边的人，何况我自己有时都不能好好照顾好自己的情绪，尽管对比起来我已经算是个乐观者了。在现在的社会里，形形色色的压力无时无刻在压迫着每一个人：家庭有所期许，教师/老板有所期许，而 TA 们自己也许也对自己有所期许（也许是分数，也许是成就/成绩，等等）。还有许许多多的事情不再一一列举。而令人感叹的是，一个人在青少年时期所受到的教育，往往不会教会他们如何去应对新时代空前的心理压力，也不能授予他们所谓“看开点”的思想境界与生活哲理。而现在的学校（不只限于狭义的学校），在心理教育、心理疏导的体制上，如同人们对于心理问题的认知一样，还有很长很长的路要走。

人们总把心理问题看作是“精神病”，进而又联系到他们所知道的精神病人做出的疯狂的举动；一个人无论对外还是对内，总觉得承认自己情绪失控/有心理问题是一件很 dirty，很 ashamed 的事情，脸上不光彩，还要损伤自己的名声，遭受可能的嘲笑甚至乎霸凌。可是这种不能从简单的咳嗽、疼痛感受到的疾病，在我看来早已经是当今社会的流行病，绝非想避开就能避开的。一方面，从自己的角度，彻底根治心理问题还真切地只能靠自己“看开一些”，至于怎么看开，我承认我也给不出什么好的方案——每个人面对情绪的洪流，都必须且只能有自己的一套解决办法。怎么去找到这套方法，一是要能直面问题，二是借鉴他人的方法。但不能完全照搬他人的路子，这样的结果往往是悲剧。（就像人生一样，适当借鉴别人的经验不失为智慧之举，但一味照搬别人的生活，那又是否是最能让自己感到有价值，比如最感到快乐、享受的人生呢？）至于对外，我以为，多播撒一点友好与关心，在不侵犯私人空间的同时尽可能地给予自己的帮助，尊重每一个人，保守每一个人的秘密，大抵也能问心无愧吧。

昨日话题的补充说明

摘自回信：

不可否认，社会生活中许许多多的问题是由当前社会的管理、处理体制产生的，但要找到一个让双方都能满意的解决方案往往非常困难。前段时间和初中班主任吃饭，她提到了在我们初中分校区的情况：年轻的老师相较于老教师，往往因为一些相对简单的事情就申请请假（也许我说得有失偏颇，但在我看来不少请假本来是可以避免的）。同桌的一位朋友说他们缺少了老一代教师对于教育的信仰。我引出本例意在说明，社会的微小参数调整绝不是容易的事情，上层给出调整的意见，下层要如何去具体实施，面对不同情况、实施人员素质的参差不齐要做何调整，并不是一蹴而就的事情。我认为社会体制的问题必定要解决，但很难希冀它在短期之内就能得到一个至少让多数人都满意的答案。

（同样的，心理咨询师也并不是万能的。我的初中副班恰好是一位心理老师，她所遇到的事情以及她个人的应对措施也说明了这一点。一方面，心理咨询师本身素质同样参差不齐；另一方面，心理咨询师同样也是人，有着常人所有的七情六欲，他们同样可能会因为共情心而被负面情绪击垮。当然昨天随笔中我想到但没有写明的咨询费的高门槛同样也是个问题。）

但这并不意味着一个人就应当去顺从。心理咨询是要让人为开心的事开心，为难过的事难过，为愤怒的事愤怒。关键在于，一个人对于这样的事情，他最真切、最发自内心、罔顾周遭环境的压力甚至乎胁迫的看法是怎样的。无论是选择默默接受还是奋起反抗（不包括过激的，已经严重影响他人正常生活的举动），我以为都应当给予尊重。我在昨日的随笔中可能也想阐述这一点：每一个人的个性是专属于这个人的标签，我们同样也有自己的个性，有独特的理解方式。对于他人的行为，我们可以从自身的角度去支持或是反对，但依然要尊重他人在合理、合法的范围内做出一定行动的权利。个人同样也要尊重自己个性的存在，不可一味的趋同，迎合大众的看法。

至于精神病/精神障碍这点，我的表述确实不够妥当，无意中表述起了污名化的作用。精神病/精神障碍同样是应当被关心被尊重的一类群体，它本身并不“dirty”，只是由于来自各类信源反映出的群体部分个体的特征，很容易让人出于不安全感而避而远之。我想到《逆转裁判》里有一个有四重人格的孩子，在凶杀案/庭审时的诸多举动都让人感受到四个人格背后的温柔。我们不可忽视恶意和危险的存在，但也不能忽视善意的存在。至于用药以及其他所谓治疗的方案，很难去评判对错，因为在这种境况下我们很难知晓患者心中的想法，往往只能求助于他们的监护人。这一点我持保留的态度，也许之后会有新的理解与体会。
