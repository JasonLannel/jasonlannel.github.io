八月随笔

发表于 2023-08-02 更新于 2023-09-01 分类于 水木录
雾海望不尽，投锚星辰间。

08/01

又补了一点图形学。实践之前学的量有点超标了，现在笔记写起来非常痛苦，感觉有好多东西要写进笔记里，啊啊啊啊啊啊啊啊啊啊啊啊啊啊啊啊啊啊啊

原计划还有两三天就要从大兴坐飞机回去了。生草的事情是家人发了视频，说是大兴机场成了大兴港，全是水，暂且不知道是否真实。（看百度好像不是真的，不会又是造谣一张嘴辟谣跑断腿吧。因为按计划回去之后有聚会，聚会群里看到家人说大兴港又好笑又难蚌）

关于名校开放的一些心声。感觉好像在面对疫情相关的事情上我的第一反应都是慢慢来，非常害怕“迅捷果断”的举动。比如去年没放开的时候希望如果放开至少花若干个月，听到突然放开的时候感觉既焦虑又恐慌。回到家期末周还是中招了。这是否也是一种犹疑（

今天运气相当糟糕，打雀魂连输四把还点了一个国士有借有还，方舟抽了 150 抽没出羊，悲。种田（小羊 CV）的声音听起来让人难过，刚开始第一反应就是 CV 嗓子不行了，没注意到原来真生病了……唉。

08/02

喝了点假酒，可能写得不是很清醒，要锅了明天会改的。

下午和朋友去中关村逛了一圈，买了点衣服。吐槽：平时非常不愿意花大钱的我今天买衣服却很果断…… 结账的时候营业员说可以买券九折，于是买了，店员说发验证码到手机上了。……咦，没有啊。结果来来回回好几次都没收到。同行的朋友也买了一张，结果也没收到。店员试了试自己手机也没收到。然后另一个店员来了秒收到验证码。究竟怎么回事呢？

原来因为她用的联通，我们仨都是移动。店员说是因为高考时放过移动屏蔽器，后来信号一直不好。从来没想过有一天移动会被联通比下去（

晚上在五道口吃饭，好多干辣椒，不小心把米饭吃完了，越吃越辣，蚌埠住了。

喜讯：可能真是假酒，两个酒量大的喝了一杯全昏了。

08/03

明天要回家了，虽然实践回来就一直在摆，但是终于有了一点点放假的实感。

有一段时间没有看几位至少我认为算是的朋友的博客，夜晚看着看着又开始 emo，唉，每天封闭在自己的小圈子里很少和别人有什么交集，即使是比较好的朋友也不过是时不时聊会天，看完博客觉得完全缺席了那些应该有所行动，至少站在身边给予 response & support 的时候。这时候又觉得自己是个以自我为中心的人。深夜真容易 emo。

08/04

坏消息：一晚没睡。不知道怎么搞的摸着摸着就到五点半了，我也不知道我都在摸什么，忘了。早上去机场去得太早，算是早到了三小时。打哈欠一直打到降落，选位置选得很好，发动机的轰鸣和呼噜声形成了完美的和声，机翼正好挡住了大半个窗户，下次还选。

好消息：大兴没淹。晴天下的大兴主打一个宽敞明亮，大的没边。坐飞机起飞前滑行了十几分钟，还以为是绕着大兴跑了一圈。总之安全润回家准备和别人吃饭。

负能量预警

不知道为什么弄了个什么清北??联谊会（我对于为什么会有这个组织抱有一点大大的疑惑），然后说到时候吃饭会有三字班的学生，可能得介绍一下自己清华就读一年以来什么感想收获。

麻的，我感觉我除了卷卷卷真一无所有了，那我介绍啥啊。负能量爆棚。前段时间有人问要不要去给暑校学生介绍就读体验，想了想我哪怕不是负面教材至少也不是正面教材，恰好和朋友出门找到理由推了，这回旋镖怎么又飞回来了啊啊啊啊啊啊啊啊啊啊啊啊啊啊啊啊啊啊啊啊啊啊！主要还是我的好朋友们都太优秀了，大一不是进组、参会就是去实习，社恐人还蹲在宿舍里啃大书瑟瑟发抖……这么一对比感觉把后辈带偏了就不好了 TAT

08/05

摆了一个白天，什么事也没做。回想起去年本应像这样摆的那个暑假，不是在练车就是在准备二招。高四结束了，摆得理所应当。

晚上联谊会学生就来这么点人，然后我才注意到拉的横幅写着家长，坏了，被坑了。

08/06

今天出去和几个小学同学吃饭。不知道怎么一点一点又加到 8 个人的，原来的群记得没那么多人啊？饭桌上明显分成了两个集团，谔谔。（以及吐槽一句，虽然我们几个人彼此之间都还处的不错，但是整个饭局感觉和我们关系不是很大，跟昨天一样。）一个个酒量都好好，我回到南方之后好像旧病复发又过敏得厉害，不敢喝，但是闻着都醉了。

08/07

在小学同学家住了一晚，中午又聚了一餐。

晚上出去吃螺蛳粉，一碗比北京便宜了 20 块，而且臭味没有北京那么浓（虽然可能有一点通风的原因），吃起来真让人感动，呜呜。虽然感觉还是高中食堂的好吃一些。

看完了之前在华为收到的《枪林弹雨中成长》。本来只是想给同学看看的，但是被误认为要送出去，总之阴差阳错就看了两个晚上（ 主要讲的是华为 00-10 年代各式各样的员工遇到的各种困难。去海外的，非洲是贫穷、没有物资，有的还有战乱。我确实很难想象他们是怎么用一种略显轻松诙谐的语气叙述出一场不过数十米的爆炸，以及一场突如其来的抢劫。之前 CTO 说开拓市场真的很难，读了一圈，一个代表处最少可能就一个代表加上一个厨师，从零信息零渠道争取各式各样的机会和经销商、运营商搭上话，与其他公司争夺价值几亿的运维合同；还有冒着枪林弹雨建基站爬铁塔扫频，在各种最恶劣的环境工作十几年，确实是难以想象的困难。何况这些故事，大多都发生在华为起步的阶段。至于跟巨头合作天天赶指标，精益求精，不成功便成仁的那种热血，从字里行间流露出来。我觉得他们所说的有舍便有得，有困难也有机遇，都是很真实的事情；假若在这样的青春，能克服自己的懦弱、紧张与恐惧，去担起一些不敢担起的工作，虽然也许只是五分钟的热度，但现在的我总觉得办法总还是有一些的，真诚、信任、细心、缜密、坚持，至少也能推开一丝光亮。摘几句有趣的话：

“时代在变，环境在变，而且变化如万花筒般炫目。但千变万化的背后，挺立的永远是‘大写的人’——人是价值创造的主体。”

“决胜取决于坚如磐石的信念，信念来自专注。华为唯一的武器是团结，唯一的战术是开放。”

好有正能量，不像是我写出来的东西。下一步大概是读一点毛选，趁着假期赶紧多读点书，学期内总感觉平时不怎么读书的我知识还是太浅薄了一些，现在有时间，怎么能放弃这么好的机会。至于图形学，可能要到 10 号才能继续更新；信号处理相关的东西篇幅不短，写起来比较耗时间。

08/08

今天写得长一些。

01

清晨七点半起来，大概是刷新了假期日常最早起床纪录。和朋友去某初高中同学 & 朋友家玩（下面为了方便，前者称朋友，后者称同学）。同学家里养有一只雄（？）鹦鹉，通体洁白，翅羽/尾羽背面是淡蓝色，一对滴溜溜的黑眼，一钩鸟喙，非常粘人，在手臂上上蹿下跳绕圈圈。自己家里没养过这样的鹦鹉，看着太可爱了一直逗着玩，可惜嘴上的谈话内容有一大半听起来不是表面上那么轻松（关于安稳与闯荡的内容）。轻松的大概是同学对于自己家的介绍（我第一次知道介绍自己家可以介绍得这么有故事色彩，不做卖房的太可惜了），以及一些关于老南宁的故事（咦，我们不是一届的吗）。三人一鸟坐看高楼风雨飘摇。

后来去迪卡侬逛，朋友准备去一趟内蒙，要购置一些装备。第一次进这种店，就像《摇曳露营》里面抚子第一次进卡力布一样兴奋。竟然真找见两列卖露营用品的地方，烤架、躺椅、睡袋、帐篷、登山杖，一应俱全。可惜没有抚子喜欢的煤油灯，也没有芝麻凛的三轮车车（车在中国不卖，而且也不会在这种店里啊喂）。借着拉了几个引体，很久没锻炼了，手上竟还留有一点茧的痕迹。不知道买手套会不会管用，但是擦镁粉抓得真稳。至于中途去华为和苹果的部分按下不表。

02

中午去了一趟水街吃生榨米粉。因为朋友很久没回南宁，特别想吃南宁的粉，我趁此机会也回味了一些过去的味道。

在我的回忆里，很久以前的水街是一条很长很长的坡，从上至下，从小吃到主食，琳琅满目，热闹非凡。那时我只有早晨坐摩托看病来这边开药时，才能因为正赶上中午的饭点在这里吃一顿。干捞、生榨粉不必再赘述；至于红豆糕、马蹄糕等等，倒可以多提几句。这两种糕往往切得方正，糕体虽然带着颜色，但透明得可以看见里面的红豆与马蹄，吃起来有点像是果冻，但糯实一些，其中有红豆的软甜，或是马蹄咔嚓咔嚓的清脆。长条状的芝麻糖、花生糖，则是糖浆填补着芝麻与花生之间的每一处空隙，咬起来硬，但是非常甜，有些粘牙。

后来水街改造，不少店铺都搬进了某栋建筑物里。门可罗雀，不复当年盛景，尽管美食依然是那样的味道。

03

去看了一眼外婆。因为在华子上学，平时没可能见她。之前在家群里知道疫情不幸地袭击了她，但幸运地是还留着一口气。虽然已经瘫痪，但总之还活着，这是我全部的认知。今天看她躺在床上，“瘦骨嶙峋”这个词非常贴切。一眼看去，你会产生这样的感觉：皮肤底下除了骨头空无一物，血管，肌肉，看不到存在的痕迹。据言也已经老年痴呆了，不过她还记得我。我妈妈说见一面少一面，虽然不管什么时候都是客观事实，但还是挺……心酸的。照顾她的亲戚也非常不容易，但我什么也没说，只是慢慢抚摸着两个手掌间夹着的那只棕色多斑的手。还是笑着好点，反正我也不善言语。

04

朋友说想给以前教过我们的一位老师送一封信（昨晚看他写信写得很晚，据他说写了一页半的 A4 纸；有趣的是同行三人都有一点写信的习惯）。为了不打扰老师，他决定放了信封小聊两句就走，于是我们在旁边买了一包信封装好就去办公室坐着等老师来。结果老师下课进来，一倒茶，一开讲，全黏在椅子上走不了了。也是意料之中。后面被邀请进去分享关于竞赛经验（其他经验给别人讲了），过去我和老师意见相悖，总觉得自己万一就是幸运之子；这次我们意见相同了，功利地搞竞赛在弱省就是死路一条。分享了一点不务正业的经验，好在老师修正了，没有误人子弟。

整个谈话太长，摘点一致意见和个人的思考吧。

1 GPT 很强，强的让人崩溃，只要有办法弄个 API 接口，什么都有可能做到。

2 以后的岗位只会越来越少，人也会越来越卷；但不再是传统意义的卷，谁更会利用 GPT，谁就更有优势。计算机相关的知识，包括大数据、深度/机器学习等在内，可能将要成为一个人必须掌握的技能或者工具。

3 传统行业都将或者正在受到巨大的冲击，交叉学科、建模（数学建模）、机器人（自动化）、保密相关等方向即将或已经成为未来的热点。至于现在的热点在哪要看前沿的论文、专利，以及投资。

（我说我有点想搞图形学的渲染、仿真方向，老师说这方面已经相当成熟了，渲染还真是，除了进某几个地儿也没去处了，最后极大概率一事无成；仿真我不清楚，但是物理不想碰啊啊啊啊啊啊。系统方向也差不多，搞算法更别提了。理论计算机一潭死水，不弄点交叉的东西怎么养活自己。我在思考弄图形学有没有机会掺和点密码学相关的东西，或者转去凹视觉，但是视觉赛道也极端拥挤且离谱。至于弄个 API 接上 GPT 让机器人理解外部世界的图形……只是个设想。）

4 有机会多去外边走走，拓宽视野，跳脱过去的、现有的思维，别困在家里。假如要闯荡，不如去外边闯荡，08环境一言难尽。

5 精进英语水平，最好是 fluent English.

6 想清楚未来十年/二十年要做点什么，做一些 creative 的事情，不要老是死磕学业。反正我绩点算是放飞，救不回平均线以上了，这话还真是挺切合的。

7 发达城市与内陆城市之间的一些差距将会逐步拉近，保守安稳一点未必不是最现实最理性的做法。

（这一点我确确实实开始在思考，留在大城市是否是个非常正确的选择。留在家乡，上一代的人脉关系也许足够一个人过上虽不富足但也小小富余一点的，有较高幸福感的生活；在大城市里，扣掉各类如租房、水电之类的消费，生活很可能相当拮据，除了奋斗还是奋斗，天天抗压。这样的生活是我所期待的人生吗？我怀疑西安高新那点人才优惠政策放在大城市里简直叫人艳羡。）

08/09

送完朋友离开从机场开车回市区。很久没摸车了，假期回来时不时就被问要不要开（不是我自己的），硬着头皮上了，万一有一天能和同学开私家车自驾游呢。感觉开高速时的我好像和大多数人不一样，一方面我敢开得比较快（大部分时候贴着限速跑），一方面又非常紧张，经常余光瞟着前后左右，生怕有什么车，死死抓着方向盘（感觉高速状态下非常不容易控制）。总体上还是安全开完了。感想就是，这车和驾校的不一样，开起来老是不适应。

下午和高中舍友在朝阳聚会。大家特意选了无恐怖元素的密室太好了，这下我就不至于表面淡定实际上怕得要命了。结果因为舍友搞各种歪门邪道，好好的密室生生弄成了一半搞笑一半恐怖的乐子故事。

谜底

德扑。很好猜吧（bushi

晚上窝在熟悉的方室里打了五六把三国杀。上次打还是在上次（聚会的时候），几乎不记得怎么打了，直接当混子；但是意外的胜率还挺高，1v1 也赢下来了。

08/10

今天不太舒服，过了。本来想写图形学的，也没写几个字，唉。

08/11

今天舒服一点，但还是晕。写了一点图形学，明天要是加把劲大概能写完信号处理了。

下午看闭社群里提到社长的事情。我一方面不愿意看到没有人接任社长的工作，另一方面却总是在踌躇——主要的问题就在社聚这一个点。作为社恐人，假如我接任了社长，必定会牵头组织社聚（毕竟我也不喜欢没有社聚的闭社，朋友出来聊天多好），但我一是没什么谈资，二也没有德扑，一言以蔽之，大概率会让社聚冷场吧。假如这样那新社友（如果有的话）来玩不就觉得没什么意思了。这样看来我还是喜欢当个小透明，麻烦别人主持社聚；那还不如拜托主持的社友来做社长呢……至于其他让我踌躇的，大概就是担心下学期压力会不会有点大，没什么机会弄社聚了。不知道社长会不会看到这一段，看起来有点“假如吉祥物也无所谓那就我来吧”的意思。

08/12

这两天是 Steam 的视觉小说节，在商店里逛了很久，往愿望单里放了很多奇奇怪怪的东西，包括：

  • ATRI
  • 逆转三部曲

七夕的饭能好吃到流泪吗

班主任和我们分享了前段时间她经历的事情。

背景：很好很优秀的老师。得了一个高级别的奖，后来七月去北京领奖的时候我被约去吃饭了（见07/13）。有奖就会有公示，比如什么奖项、什么理由颁给谁，比如每个人的履历、事迹等等。老师的事迹也放上去了（比如花了几年时间一直去坚持到城中村给留守儿童上课、为了备课理解桥梁结构一直问到大学教授），每件事情都是非常真实的（至少我是很相信这一点）。

故事：后来每日头条放了一篇文章，标题写《XXX中学教师XXX事迹材料》（注意，没写获得什么奖项），然后只丢了履历、事迹和照片上去。后来评论区就出现了各种各样的声音，比如怀疑事迹是否真实，履历和职称是不是靠着某种不可言说的关系，以及任何荣誉都不如学生肯定，等等。总之慢慢地人身攻击多了起来，成了网暴。

后来学校里不少老师也看到了。按照我们老师的叙述，有一部分教师开始用非常异样的眼光看着她。朋友/有一面之缘的人之中有关心安慰她的，也有在一些会议上大声地关心到“XXX，你最近成了红人哦”。

老师说那段时间多少有些抑郁。有些奖，她自己无所谓能不能得，因为以她之前获得的荣誉，早就不需要更多的奖来证明她的优秀了，只是这些奖项对于其他人很重要。老师如今负责学校许许多多的事情，忙得焦头烂额，没有时间去相亲。有人传出她已经有了孩子，一时间闹得沸沸扬扬。为什么会这样呢？老师说某领导给她分析，说别人觉得她已经拿了奖，说几句又有什么呢。

评论：我当时说，这种话根本就没有逻辑，“欲戴王冠，必承其重”，这个“重”难道不是说戴上王冠就要有配得上王冠的行为举止？（后来吃完饭回家的路上我想了很久，感觉我这么说也不对，戴了王冠的国王怎么说也是个人，做出了正常人的举动（比如生病休息）就要被人抨击么？）

同桌的朋友都很生气，表示要到评论区骂回去。但我觉得实际上没什么用。老师说人们觉得正面的消息没什么意思，不如负面消息吸睛，这句话在如今确实很有道理。他们总能从字里行间（包括可笑的 IP 属地）去找到反驳你的若干个理由，于是你也成为众矢之的。

我后来的想法是：也许是因为政府公信力下降了吧。

这么多年来，不管是扫黑除恶行动，还是铺天盖地的各种新闻，算是知道了政府内部也有着许多见不得人的勾当。包括电视剧里放的也是如此。这些实际上也在百姓心中种下了怀疑的种子，公信力就慢慢下降了（当然，还有很多原因，不止这一点）。不管什么事情，都要去怀疑批判一番，不怕冤枉好人，就怕冤枉坏人。网上的环境老实说真的很脏，净土都是小块小块的，总能找到拉满攻击性的、充满谎言的地方。反正网上评论又不犯法，赶紧评论怀疑一番，有调查报告再怀疑一番。要看到有人维护，先怀疑是不是请来的水军。总而言之，怀疑就对了。再煽风点火，就成了网暴。

有时候我想着，自己毕竟不懂事情的全貌，于是便噤声等着专业的人来回答。可问题是，那些比我更不了解的人，用那种一看就是扭曲的、煽风点火的言语，误导了许多人。这时候又觉得一言不发又是错误的选择。唉，挺矛盾的。

“你怎么都不愿意花钱买个梯子啊”

我总是在很奇怪的地方抠门，比如梯子，比如加速器。结果就是我基本不用梯子，用的时候也是找的某个免费梯子，总之就是很烂。

后来有朋友找我说要不要合租一个。想起来舍友有梯子，于是我问他用的啥，我也买个试试。结果他说他学弟教他搭了一个，只要有域名就 OK 了。听起来好棒。于是麻烦他发了个视频教我搭，总而言之就是：

  • Azure 学生认证完了可以弄个服务器玩，每年有 100 dollars 的额度；
  • Github 学生认证可以去弄个免费域名；
  • 想办法把域名解析啥的搞定，弄弄端口啥的，配置一下 dashboard 方便管理。

于是就得到了免费的梯子，感觉也不慢，够平时用了。
