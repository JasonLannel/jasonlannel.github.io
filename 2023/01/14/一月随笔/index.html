一月随笔

发表于 2023-01-14 更新于 2023-01-25 分类于 水木录 阅读次数：
记录一月的生活。好奇怪，这是不是我第一次在博客上写日记？

01/13

明天是跟我非常敬重的两位老师见面的日子。一位是我高中的信息竞赛老师，指导了我信息竞赛的全部生涯；还有一位是我初中的班主任，教给我许多人生的智慧以及道理。晚上想着选点什么礼物好呢？于是翻看自己的包里，除了几本没有拆封过的笔记本什么都没带回来。看来以后还是得随身带着明信片以备不虞！ 希望明天能顺顺利利，在某些桌上好好表现，别社恐中途犯了。

唉，什么时候能解决社恐呢？我都不知道为什么会有社恐，但它就是发生了。

今天看普物非常无聊，打开 Github 翻了翻，意外翻到了不少我认识的人。果然 Github 用的比较多吗？一边看大佬们的归档一边想，啊啊，原来他们以前都做过这样那样的事。又回念起我的 OI 生涯，实在是很贫乏的一段日子，一头闷进竞赛里，许多当时知道的不知道的都没有去尝试过。当时最大的梦想还是在竞赛中拿点奖励（特别是洛谷月赛的笔记本！），但是到最后也没有实现。

突然打算写点日记也是因为翻到驰雨老师的日记博客，想到自己高一时那个不知道写给谁看，最后下落不明的日记，心里眼泪在哗哗地流。博客果然还是充实点好吧！而且我也需要一些方式处理我的情绪，把各种我心里想说的事吐露出来。至于有没有人看，大家读起来什么感受，那已经不是我该在乎的事了。只是希望我能多坚持一点吧。

过几天宣讲，开始写自己的专业介绍了。写的时候有一种奇异的刚刚收到录取通知的感觉。可能是进 IIIS 对我而言实在太过震撼了。

（关于我去茶园当茶农这档事里记载了我进入 IIIS 的经历。）

一些可能有点沉重的东西

惊险的故事每次想起来都非常后怕：如果我没有拿到银牌，如果我不是恰好踩了清华的线，如果我笔试没有侥幸通过，我现在会在哪里？

我不知道。

也许我还住在南区，和我机航动的两位好兄弟们忙着画图，兼修计算机/茶园的课程；

也许我在强基里读着一个我不怎么喜欢的专业，每天为了未来疯狂地卷卷卷；

也许，我根本到不了清华园，好一点能靠着我自身条件去浙江或者上海读计算机，差一点与计算机永远告别，而据我所知，不管是好是坏，结果都未必如我所想。这个疙瘩也许会伴随着我走完这一生。

这段也许看起来非常凡尔赛，毕竟他们都没有发生。但实际上这些故事才是最有可能发生的。我认识的 OIers 有多少人会走向其中的一个结局呢？我不知道，但绝对不在少数。IIIS 凤毛麟角，圆梦的人就那么多个。我能意外上岸，现在想来不哭上一晚上实在是说不过去。

那么，现在我满意吗？

心里冒出来的第一个念头竟是：我不好说。

在这里压力很大。我虽然踏进了门，但我也清楚相较于各位大佬，我的天资、我的知识，未必有平均的水平。我要做2小时的事，别人1小时就能做完，做的又快又好。我所不能做的，或者希望去学的，别人早就一溜一溜地从天说到地给你讲个透甚至拿着成果给你看了。而且我也不是一个很会思考的人，并不能像其他人那样，问出各种我所想不到的问题。上完课甚至不知道自己还有什么疑问，这就是从开始读书到现在的实际情况。这一点也不好笑。在这里，我接受自己的平庸，却不能甘心承认自己的平庸。我还在追求着优秀的绩点，一学期奋斗下来被期末考干掉丢了绩点，我心里除了难受还是难受。尽管我最后成为末流的概率并不算小，由于身体原因我周围也会劝我放弃那点绩点的幻想，接受自己比不过别人的现实，但我无论如何也不愿落到那样的境地。忽然想到自己当初在特训班里偶尔还能争到前三，最差也是班级平均，感觉有点像在看乐子，哈哈。

只是我还不愿太早放弃。毕竟我还是爱着信息学的。哪怕我没有能力，我也希望能尽力参透里面的奥秘。我希望能把这件事做到我的最好。明明都来到国内最好的，至少也是数一数二的信息学殿堂了，怎么能在这里放弃。以前我希望能看一眼天才们的殿堂里的世界，现在我看到了，不愿离开不也是很自然的事吗。只希望现实能慢一点，让我代替那些没有成功的人们在前面多跑一点，在殿堂里多看一点。

某种意义上我还是很想做一名老师的。

你开头这么沉重，后面人家还怎么看啊喂！

01/14

今天去参加鸿门宴聚会了。突然间就有点感伤，不过这些话明天再补写吧。

然后你就在一周后的大年初一来补是吧？

早上和 H 老师以及其他两位好友爬青秀山。上次爬青秀山还得追溯到高考刚结束的时候。那次参加的同是我们四人，一路上 Z 一边走一边估分，我因为早就估完，分数算是中规中矩，心情倒是比较轻松。三年来没有登上过的龙象塔也是在那一次终于圆梦，九层的宝塔，站在顶上十几平米的小空间里远眺巨大的三明治：整座城市是一片钢铁的面包，夹着中间黄棕色的滔滔大江，又以一片连绵的森林的浓绿作底。踏上去的那时只感觉很累，但现在一想，这正是饮酒赋诗的大好时机，端起一盏，横卧于楼台方窗之间，轻缓缓地吐出几个字来。虽然我既不会饮酒，也很久没有赋诗所需的那点想象力了。

说道爬山有几件趣事。我们打算重新登上市区最高峰，那里是一座塔的遗址。按着导航给的地图随机漫游，结果走着走着，发现是军事管理区。怎么 5A 景区里会有军事管理区啊喂 然后四个人就一边聊一边沿着景区的石头小道走，结果怎么走都会回到军事管理区；到了最后我们甚至看到了突兀的篮球场，直接对全场所有人施加了沉默。还有一个比较离谱的：我们爬山的时候听到很响亮的吱吱声，老师眼尖，看到路对面树上有松鼠。山里面松鼠并不怎么常见，所以大家都很兴奋的拿起相机走。结果走近一看，乐了啊，竟然是两只松鼠！我们所有人都尴尬而不失礼貌地笑了，竟然在这里可以一睹自然界的奇观，生物休养生息所必不可少的一部分。然后某人就这样录完了从开始到结束的全过程，并且存在图库里的私密空间。我们笑着说不怕被孩子看到吗

晚上是和初中班主任吃饭。每次来聚会的来来回回就十多号人，说到底发起人也是这十几号人，大家也没有什么放不开的。

（然后我就一路被损死，被评价说出来的话太过无趣，不像是自己教出来的学生。唯一有点安慰的是另一个上华子的也被损死了，但是我比起来又远远不及。合着我就是来吃牢饭）

席上听到了其他同学的生活细节。感觉所有人自从从十四毕业以后各奔东西越来越活成了自己当初（大概）不愿活成的样子。而且一些同学自己遭受的苦难也在会上吐露了出来。除了非常普遍的光速恋爱光速被甩之外怎么这也能普遍的！！！为什么我就没有！！！，还有一些诸如校园暴力这样非常压抑的惨案。好在到场的大家都从阴影里走出来（至少表面上是这样）不开玩笑地说，没走出来的多半也不会来。想起当时初中发生的各种各样的事，生活的残酷就这么压在眼前。除了叹息和严肃之外能做的也没有多少了。

后来班主任主动说要送我，然后我才知道我们敬爱的老师是故意损我们两位华子人，要不然大家来聚会还要分个高低实在太没有意思了。也怪我没想到第二层，哭死，这波被损是上课了。 又获得了很多饭桌上没有讲的道理，让我们对于大学生活以及在那之后的未来有了更加深刻的认识。还有很多要克服要实现的东西啊。

01/15

昏睡了一天，感觉没什么好说的。

摸鱼。摸鱼。摸鱼。不行啊，你不能在这样颓废下去了！摸鱼。

但是，前面忘了中间忘了后面忘了。

01/16

要补的东西都挪到今天；但是今天做了一整天的 PPT，等后天给学弟学妹们做经验分享，14号的日记又要挪到明天。没有时间规划就是这样的。

关于专业介绍，就我个人而言不太想讲。倒并不是说无话可讲，展开起来能讲很多，只是总觉得没有这样的必要。我觉得我朋友说的很精辟：讲这个有什么用，又没几个人能上。而且哪怕真够到了，我觉得没有太多热爱真不至于要来茶园，没必要受这种苦。想到去年暑假时同校的茶园学姐劝退我时，我还两眼放光觉得自己能克服诸多困难，啊哈哈，勇者如今也成了恶龙。下了水就知道水里是什么了。至于就读体验写起来真是一种折磨，明明里面每一张图都是我拍的，都是我亲身参与经历的，但是总觉得不太真实，明明天天闷在教室或者宿舍的日子才是日常。现在看起来感觉非常美，好像大学生活充满美好！但是实际心里已经在各种吐槽了。（就像我从南区搬到紫荆时的心理活动一样。）

理综的学习经验也是由我来主讲。小队里我的理综成绩是数一数二的，交给我来做非常正常。（虽然我很想这么说，但开诚布公地说：实际上是因为当初开会时不太想发言，结果就剩文科类的语文/英语和理综了。可我语文常年班级末流，英语又是高考被爆，还不如讲讲爆种的理综。）刚开始想着随便讲点算了，但是等到开始动手，我意识到自己理综学习就是个异类，整个高三写下的笔记寥寥无几，最多也就写点自己常错的东西，剩下的日子就是刷题卷（三声）卷（四声）。做题顺序也是随心所欲，今天先从客观题写，明天可能就从选做题写；看着感觉难了就直接跳过好几道题目找容易的题做。然后滑稽的事情发生了：我抓着我熟识的几个高中朋友拜托他们教我学理综。他们表示非常惊诧：你不会是要复读吧？

之后水了很多不限于理综的我觉得还算重要的东西。感觉其他的队员也会讲吧……到时候大家看着我把PPT从40分钟删到20分钟，我已经等不及了！

01/17

今天给自己提前买了新年礼物，就是这个！

趁着打折终于把计划通买下来了。钱包在流血呜呜呜。然后试着玩了玩 Xodus 和 Freedom Dive，我突然意识到自己被骗了。这是人打的谱子吗？

粥游今天也大更，登临意终于是开了。官方的特别映像感觉做的很不错，无论是作画还是配音都非常舒服。有时候会想要是番剧的水平也有这样就好了，虽然很不切实际。抽不出想要的角色这件事暂且压着不表。

等到明天宣讲完了打算着手计划去杭州跟同级的几位大佬（抱住大腿啦！）参观某地的事。我心里非常期待这趟旅程，毕竟在杭州埋藏着我太多的时间。还想见见我杭州的老朋友，能一整天谈天说地的交心的朋友屈指可数。也许到了杭州会聊一聊过去的事吧？

突然想聊聊重建博客的事。

最开始建博客是拿来存我的高中复习资料的。作为 OI 退役老选手以及特训班的一员，高三我可以经常跟电脑打交道，顺手把博客建好后在博客上存了很多语文学习的作文素材与经验总结。当然也只有我一个人看。后来开始用 Github 做正经事，想着留着这些东西也没用，一次性全部删掉了。

前几天突发奇想，Github 上是不是可以找到我认识的一些人呢？不查不知道，高中同校 OIers 都被我查了个遍。然后顺着奇怪的社交链条一步步找到了几位社友。趁着这样的机会修了修自己的主页介绍，把博客重新启用了。

至于开始写随笔的事，其实是看了驰雨老师的博客想起过去高中写的一些见不得人的遗失了的日记。我自己没有记事的习惯。但最近我开始意识到一些珍贵的记忆正在我的脑海里变得模糊。抱着这样的一种愿望把日记丢到了博客上。以及我自己在个人博客里开理发店也不会有人说我，反正我也猜不到谁看了，无所谓了。缓解缓解压力多说点平时在台面上不能说、不想说的话也对自己身心健康有所帮助。

好像又把14号的咕了。这……

01/18

宣讲圆满结束！社恐犯了，但是好像发抖的声音经过网络的加工没展现出来，没人看出我慌了。 感觉有没有稿子的我是两个样，有稿子念（指介绍茶园）我能展现出最优秀的pre，但没有稿子（指分享经验）我直接退化成小孤独。啊哈哈……之后母校表白墙上好像有评价说是个温柔的学长，你确定不是社恐加分吗

抓到图了，别人眼中的我是这个样子吗（逃

但是很感动，这就是我希望我的分享能起到的作用啊qwq 毕竟离开高三太久，很多细节早已经忘得精光，作为学长能做的只有一点点贡献还有鼓励了。至于内容对大部分同学适用，那是因为上清北的就那么多人，晚上来听宣讲的甚至把腾讯会议间的人数上限给爆了，显然来听的更多还是最后来不到清北的啊…… 而且事实上本来能上清北的同学到最后就是各种细小的问题出了锅，所以这些实际都是真正的干货。

自己的部分结束之后我只需要负责截图积累材料。非常无聊，于是跟 Sweetlemon 从东到西聊了很久，慢慢地扯回到了学习上。下学期的紧迫感又回到了我的脑海里，那个让人害怕的学分和课表，明天开始不努力不行了啊！然后继续摸鱼一路摸过一月的事也不是没有可能

准备开始温一温下学期的课。好似回到了去年暑假那些奇怪的日子。

01/20

玩了一整天接力肉鸽，摆了（

今天在整一点资料，等着明天回老家之后看。上上下下把各种网站（贵系科协，csdiy，etc.）翻了个遍。To-do 文件夹里突然就多了好多东西，希望能肝完一半吧。

成绩出了一大半，看起来还算满意，除了自己院开的都非常不错。上平均线我就很满足了……比较让我惊喜的还是写沟，没想到能拿到 A。想当初我想着能水过就行，没想到到了期末写沟已经是我最期待的一门课了。老师又好给分也好，任务（据说）也比其他的写沟课轻松，之前我自己找的选题也很有趣，没有什么理由不让我对写沟动心。编入门人均A+，不是也挺好的。

听说人工智能入门向下调分，但是我没选，好耶

01/21

比较忙的一天，一晃眼就过去。不出意外初一早上会把这一部分补齐。作为补偿把除夕画的神魔东西放上来供大家一乐。新年快乐！

谢谢，现在就由初一的我来续写除夕的我遗留的工作。

原片见2022华灯宴单品 搞笑东方日和 名侦探爱丽丝酱。

下午坐车回到乡下老家，看到微信群里四人组在发红包，突然想到最近有自己制作定制红包的活动。作为一个感性的人，我一时头脑发热，定制红包用的图当然得是我画了！于是胡乱画了一堆东西，刚开始想画的 Amiya，最后跟蔚蓝档案糅合在一起造出了全新兔子。个人最喜欢的是 Mika 兔，只是可惜不会上色， style2paint 终究不能让那片朦胧的面影变得清晰。等到制作好了，又觉得自己做出来的东西是一团不可知的云雾。但是五六个小时（是的，这就是我在忙的事情）已经在数位板的涂涂画画中逝去了，那也就顺其自然吧。不会画画但又喜欢绘画的我就是矛盾的集合体。

画完绕着屯口转了一圈，屯里好多家正围着吃烧烤。我跟着爸一起，本来只是想看看烧烤得如何，如果可以偷偷顺一点东西吃，忘记了华子大学生的名声早就不允许我能自由活动了。于是开始不断地听着我自己所听不懂的方言应和点头打哈哈。又实在不好意思跟大家说别戴着华子的镜片看我，我只是个普通苦逼大学生这样的话，总觉得这种话由我说出来实在是太狂，这个身份早就让我没有下山的可能了。手足无措的我在寒风里冷冷地无奈。

01/22

起来已是晌午。

今日雨雾朦胧。从窗里抬头望向老家背后的大山，白色的云卷遮盖住半山腰，山顶已经隐没于烟雨之间。现在想来，这多像是去年舟舟那座不存在的三山十三。也许山顶之上是一道艳阳，一片云海，一座亭，一挑古灯。至于年兽，也许是在山顶上睥睨着凡间五色的烟火吧。至于为何不在人间，也许是因为年味淡了。

凌晨忙完给几位我非常重要的朋友发完祝福，这种感觉也越是强烈。抢红包的活动依然是有，拜年纪（祭）、春联（春节联欢晚会）也依然是有，烟火放得比去年前年更旺。但是总感觉一种精神上的冷清。现在想来，肯定是一些非常重要的东西渐渐失去了。小时候看的还是春联，那时老家还没有重建，冰冷的屋子里大家靠在一起看春晚，打牌的，吃零食的，摔炮的，一片祥和。之后有了拜年祭，入坑动漫的我那时对它充满特殊的感情。四迹、艾叶（铁心博弈和五十在我心里有着 excellent 的地位）、初音未来，还有每年的 SPICE 的预告，诸如此类的宝藏构成了我对于二次元的美好的想象。之后是也许东方众会比较熟悉的东方华灯宴，我总是刷了又刷，反复咀嚼，想象着同人所能做到的宴会是如何的热闹与欢喜。然而到了今年，我已经寻不到那个通往宴会的道路了（华灯宴停办了，而宣布停办的企划人甚至没大我几岁……一个年代的人可能会更有共鸣吧）。有一股，甚至是好几股洪流把年味不断地从案板上洗去，年夜饭所残留的食材的气息随着洪流也一并冲走。洪流姓何名甚？我不知道，又或许只是不愿说出口那些阳光的词汇。

惆怅也只是为了那一场必不可少的宴会，其余倒是人间依旧。和老哥玩了半个下午的手柄，讲的是修改过后的三国故事，像素游戏让我回忆起看 C 玩八方旅人的日子。最后感慨，我果然还是玩不来这类游戏。

晚上拒了几次，最后因为拗不过小孩开了一听可乐，从饭桌上下来跑到冷清的大厅里慢慢地灌。有时汽水对我来说就是酒，喝下去的是甜味，但是总会转化成一种虚幻的眼泪，心中不自然地冒出各种各样的，平时所不会想的东西。全身的神经都在某一时刻变得酥麻，从脚底跑到脊椎，最后消失在头顶。现在想来，我喝汽水最多的时候大多也是我自己一个人的时候，一个人喝闷酒，排解心中的压力；或是高兴了，一瓶北冰洋或者雪碧就是自由翱翔的小洞天。只是现在成人，做什么事情都会越来越不自由；自由的时间，随着那一点二氧化碳弥漫在空气里轻轻地在眼前我所看不见的彼岸飘荡。

01/23

深夜一个人看了春节档特供艾导电影《玛特罗什卡》，观后自觉地交了四个硬币。

剧透

我觉得套娃的这个意象非常精妙。看似紧密相连的火车车厢实际上在时空上是完全分隔开来的，而实际上又作为雯的一生紧密串联起来。从父亲金向舟的黑化，到她自己的光暗的斗争；从沧海的父亲曾炎潜伏毒窝之中的自我救赎，再到丈夫曾沧海为爱别离在外奔波养家糊口与蓝围巾的陪伴，这一连串的套娃是非常巧妙且细致的，正如艾叶自己所说的：这是一个叙事性诡计。这个诡计无疑是非常成功的，而那些非常认真的观众也可以从许多蛛丝马迹中自己慢慢揭开这层面纱。最后的那个关于对错与得失的内容，以及最后雯最后作笔录的一段，都让整部作品在反转中活得非常自由与彻底，升华成了一部不止于合格的优秀的“电影”。

我个人觉得这种形式是所谓正能量的最为适合的一种表现形式：正能量应当是在最后自然而然地输出，它真实作为这场艺术演出这一整体的其中一个有机部分出现；它可以是全篇的主旨，可以是用于升华，也可以只是添个彩头，但不是这场演出最关键的那一部分，不是限制这场演出的紧箍咒，也并不是必须要开门见山才能算说清的东西。它的引出应当是不生硬、不死板的，这个过程没有俗气或者高雅之分，没有刻意的矫揉造作，只是一种引导观众感情自然流露的一个过程。

我也接受一些其他的说法：有些艺术形式并不是为宣扬正能量而服务的。今天在刷视频的时候偶然刷到了赵本山先生的采访。他说，小品就是用来给大家乐的；一年365天有那么长的时间都在受教育，剩下的这一天也教育不了什么。有人拿花鸟画作比，花鸟画里面欣赏不出什么正能量，但并不影响它的艺术价值。我是比较认同这种说法的。想当初小品就是春联上专门给大家乐的节目，现在却不能让大部分人从头到尾都能乐。时代发展真快啊。

下午从老家回城，本想着自己开一段路，但是最终还是没说，尽管别人已经帮我把话题都抛出来了……现在对于自己还会不会开车这件事抱有很大的疑问，万一高速路上失误了怎么办呢？这时候还是能虚就虚吧。

以及向大家推荐家乡的一种炸饼。看起来是饼，外观上的那层炸皮比较像是天妇罗；中间露出不清楚是酸菜还是梅菜的馅。吃起来是一种很奇异的味道，总体是偏咸的。这就比较对我的胃口了，毕竟我吃东西不知道是不是练出来的，所谓对胃的味道总是比其他人要咸一点点。于是我自己煮面的时候就被家里人吐槽放盐放得太多了，但对我而言确实刚刚好的味道。

不太想展开的东西

晚上又被问起当初清北学生集会的事，唉，刻意不提的事情又被重新提起了。总觉得大家对于集会有着很多错误的认识，好像集会就是什么非常见不得人的事情。我个人觉得集会也是一种表达方式，只是这种表达方式应当能自证它的正规性、合法性，有明确的目的或者诉求，其目的/诉求也应当是合理的，克制的，爱国的。好像说得太多了一些，还是把它就这样忘却吧；现在已经放开，就让它在历史的尘埃里尘封着。
